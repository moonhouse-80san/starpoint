[설치 방법]

1. 다운 받은 모듈을 설치 - 관리자 설정 없음

2. 게시글 별 평점 모듈을 적용할 게시판 스킨의 파일에 아래의 코드를 작성 

(예 : /modules/board/skins/sketchbook5)
